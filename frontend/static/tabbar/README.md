# TabBar 图标说明

请在 static/tabbar/ 目录下放置以下图标文件（建议尺寸：81x81px）：

## 必需的图标文件：

1. **home.png** - 首页图标（未选中）
2. **home-active.png** - 首页图标（选中）
3. **health.png** - 健康图标（未选中）
4. **health-active.png** - 健康图标（选中）
5. **medication.png** - 用药图标（未选中）
6. **medication-active.png** - 用药图标（选中）
7. **notification.png** - 通知图标（未选中）
8. **notification-active.png** - 通知图标（选中）
9. **profile.png** - 个人中心图标（未选中）
10. **profile-active.png** - 个人中心图标（选中）

## 图标建议：

- 首页：房子图标
- 健康：医疗十字或心形图标
- 用药：药丸图标
- 通知：铃铛图标
- 个人中心：用户头像图标

## 图标颜色：

- 未选中：#666666（灰色）
- 选中：#007AFF（蓝色）

## 注意事项：

- 图标必须是PNG格式
- 建议使用透明背景
- 图标尺寸建议为81x81像素
- 如果没有图标，可以在pages.json中暂时移除tabBar配置
